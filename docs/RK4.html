ODE_Solver¶

ODE_Solver.Solver(dzdx, x_i: float, x_f: float, y_i: float, dydx_i: float, h: float = 0.01)¶

ODE Solver

Solves second order ODEs with Runge Kutta 4th order method

Parameters

  • dzdx (function) – This is the slope of z (y’) with respect to x.

  • x_i (float) – Initial value of x

  • x_f (float) – Final value of x

  • y_i (float) – Intial y condition

  • dydx_i (float) – Initial y’ condition

  • h (float) – This is the step size from current value of x to its successive value

Returns

It contains all the values of x, y and z (y’) from all the iterations

Return type

numpy.ndarray

ODE_Solver.Solver_with_AdSS(dzdx, x_i: float, x_f: float, y_i: float, dydx_i: float, h: float = 0.01, min_diff: float = 1e-05)¶

ODE Solver with Adaptive Step Sizing

Solves ODE with Runge kutta 4th order method. Adaptive Step Sizing helps in speeding up ODEs with lot of flat slopes in its solution and helps in collecting necessary datapoints, discarding fillers. It adapts it self to the curve by changing the step size appropriately. Adaptive step sizing can slow down simple ODEs and ODEs with lot of steeper slopes but a trade off with rich data.

Parameters

  • dzdx (function) – This is the slope of z (y’) with respect to x.

  • x_i (float) – Initial value of x

  • x_f (float) – Final value of x

  • y_i (float) – Intial y condition

  • dydx_i (float) – Initial y’ condition

  • h (float) – This is the step size from current value of x to its successive value

Other Parameters

min_diff (float) – This is the minimum tolerance of difference of y between current step size and its half or double step size

Returns

It contains all the values of x, y and z (y’) from all the iterations

Return type

numpy.ndarray
